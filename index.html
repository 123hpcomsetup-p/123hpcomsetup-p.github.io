Download & Install HP Smart Drivers and Softwre


  • Set up your printer to PC and turn on the complete system.
  • Open a browser and visit 123.hp.com/setup.com website.
  • Enter the HP printer model name number (see on the device front).
  • Select a driver, then click Download and wait for most recent HP setup download.
  • Locate the downloaded file and run the installer.
  • Double-click on the setup to start the installation.
  • Read and accept the user agreement license, then register your product.
  • Follow prompts, install cartridges and try printing.

How to connect a printer via a wireless network?


Going through 123.hp.com/laserjet provides wired and wireless, both Hp printer software download options, hence if you have hp wireless printer, connect it with following steps:

  • Turn on the hp printer and connect to home WiFi also, see the available WiFi networks on display.
  • Choose one network to manage Wireless LAN Settings, then locate the SSID from the router.
  • Enter the password, open printer settings, and add your printer from the window setting.
  • Once done, visit 123.hp.com/setup to download and install the HP printer drivers.
